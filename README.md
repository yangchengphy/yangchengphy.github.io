# Cheng Lab homepage

A clean multi-page static website for GitHub Pages.

## Launch on GitHub Pages
1. Create a public repository named `yangchengphy.github.io`.
2. Upload all files in this folder, keeping `index.html` and the page files at the repository root.
3. Keep the `assets/` folder in the same repository.
4. In Settings → Pages, deploy from the `main` branch and `/root`.

Before launch, update the UMass Lowell email address on `contact.html` and any news items.
